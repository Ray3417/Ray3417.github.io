## 三、视觉风格特征

### 3.1 整体氛围
- **治愈系 / 日常感：** 不追求戏剧性构图，呈现安静、平和的日常场景
- **竖屏壁纸比例：** 9:20 的瘦长比例，强调上下空间（天空/地面）
- **日本动漫美学：** 柔和渐变、清晰的天空分层、细腻的云朵和光影

### 3.2 天空处理
- 上蓝下暖渐变（地平线处天空混入地面暖色）
- 云朵通常用高光白色/淡蓝色勾勒边缘
- 黄昏场景（Image-2）天空呈粉紫渐变，带有日落感

### 3.3 地面/建筑处理
- 以暖橙棕、土黄、暖灰为主
- 建筑轮廓较为简洁，扁平化处理
- 植被用深绿 + 暖色叠加表现

### 3.4 光影特征
- 柔和光影，无硬阴影
- 高光区普遍过曝偏白（Anime 风格常见）
- Image-4 呈现明显的黄金时段/室内窗光效果

## 四、网站设计应用建议

### 4.1 色调选择
- **首选暖白/米黄背景** `#FDF8F0`（与插画整体明度匹配）
- **蓝色仅作点缀**（天空色 `#0060e0` 用在按钮、链接高亮）
- **暖棕色系作装饰**（分隔线、hover态、图标底色）

### 4.2 不适合的方向
- ❌ 将暖色调原图强制转为蓝色主题（丢失原图氛围）
- ❌ 使用原图作为全屏背景（分辨率 1080×2400 拉伸到桌面会模糊）

### 4.3 推荐的设计方向
- ✅ **CSS 渐变重构：** 用 `#FDF8F0` 米黄底色 + 从插画提取的蓝/橙/粉做局部渐变
- ✅ **保留原图做局部装饰：** 将原图裁剪为合适比例的小区块（如 Hero 顶部天空切片）
- ✅ **噪点/纹理叠加：** 用 CSS noise 模拟插画的手绘质感
- ✅ **色彩提取后应用：** 天空蓝、暖棕、土黄均可用作 Tailwind 自定义色板

### 4.4 CSS 变量建议

```css
:root {
  /* 背景 */
  --bg-cream: #FDF8F0;
  --bg-warm-white: #ece8d2;

  /* 天空蓝 */
  --sky-bright: #0060e0;
  --sky-light: #5ba4ee;
  --sky-mid: #4284af;

  /* 大地暖色 */
  --earth-warm: #b8772e;
  --earth-soil: #b17e2d;
  --earth-muted: #975b43;

  /* 暮色 */
  --dusk-pink: #eed1d5;
  --dusk-rose: #c0817e;

  /* 高光 */
  --highlight-cream: #fce4d0;
  --highlight-cold: #d6f0fb;
}
```
